================================================================================
六、任务解答命令（完成题目要求）
================================================================================

=======================================
任务1: 配置命令本地保存并发送到 Syslog 服务器（隐藏密码）
=======================================

R-WEST 配置:
enable
configure terminal
!
archive
 log config
  logging enable
  hidekeys
  notify syslog
  exit
 exit
!

说明：
- archive log config: 进入配置日志模式
- logging enable: 启用配置日志功能
- hidekeys: 隐藏密码等敏感信息
- notify syslog: 将配置变更通知发送到 Syslog 服务器

=======================================
任务2: 配置 OSPF LSA traps 发送到 SNMP 服务器
=======================================

R-WEST 配置:
enable
configure terminal
!
snmp-server enable traps ospf lsa
!

说明：
- snmp-server enable traps ospf lsa: 启用 OSPF LSA 相关的 SNMP traps

=======================================
完整解答命令
=======================================

R-WEST:
enable
configure terminal
archive
 log config
  logging enable
  hidekeys
  notify syslog
  exit
 exit
snmp-server enable traps ospf lsa
end
copy running-config startup-config

八、知识点说明
================================================================================

=======================================
Archive Log Config 功能
=======================================
- logging enable: 启用配置变更日志
- hidekeys: 在日志中隐藏密码等敏感信息
- notify syslog: 将配置变更发送到 Syslog 服务器

=======================================
SNMP OSPF Traps 类型
=======================================
Router(config)#snmp-server enable traps ospf ?
 cisco-specific  Cisco specific traps
 errors          Error traps
 lsa             Lsa related traps
 rate-limit      Trap rate limit values
 retransmit      Packet retransmit traps
 state-change    State change traps
 <cr>

本题要求配置 lsa 相关的 traps
